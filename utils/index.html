一个 JavaScript 公用方法集合 

设计规则

  • 零依赖
  • 不会抛出异常.
  • Never assume anything about input value(s).
  • 总会返回一个合理的默认值

命名规则

Prefix 前缀Condition 返回
check返回一个布尔值
generate返回一个新数据
toReturns coerced input
不返回数据的函数没有前缀

与这种优雅的变量命名没有冲突，例如：

const isUnique = checkUnique([1, 2, 3])

Install 安装

npm i @balac/utils

checkUniqueArray

检查数组项是否唯一。对于非数组的任何值将返回 true。

import { checkUnique } from '@balac/utils'

checkUnique([1, 2, 3]) // true
checkUnique([1, 1, 2]) // false
checkUnique('not array') // true

generateIntegersArray

生成一个从给定数字到另一个数字的整数数组。

import { generateIntegers } from '@balac/utils'

generateIntegers(1, 6) // [1, 2, 3, 4, 5, 6]
generateIntegers("abc", 6) // [] - 有一个参数不是数字

checkRunningInBrowserEnvironment

检查当前代码是否在浏览器中运行

import { checkRunningInBrowser } from '@balac/utils'

checkRunningInBrowser() // true

sleepMisc

暂停当前函数

import { sleep } from '@balac/utils'

await sleep(2000) // 暂停 2 秒
await sleep("abc") // 传人非数字的参数，暂停 0 秒

toNumberNumber

将传入的参数作为数字返回，不能转换成数字则返回 0

import { toNumber } from '@balac/utils'

toNumber('this is text') // 0
toNumber('3') // 3
toNumber(0.4) // 0.4
toNumber(new Set([])) // 0

toOrdinalNumber

如果可能，将数字转换为其序数字符串形式，否则返回第 0 个。

  • 1 -> 1st
  • 4.2 -> 4.2th
  • notANumber -> 0th
import { toOrdinal } from '@balac/utils'

toOrdinal('this is text') // 0th
toOrdinal('3') // 3rd
toOrdinal('11') // 11th
toOrdinal(3.14) // 3.14th

checkCompactCaseString

检查传入的字符串参数是否包含空格，非字符串、带空格返回 false

import { checkCompactCase } from '@balac/utils'

checkCompactCase('thisiscompact') // true
checkCompactCase('this is not compact') // false
checkCompactCase(123) // false - input is not a string

checkLowerCaseString

检查传入的字符串参数是否包含大写字母，包含大写字母、非字符串返回 false

import { checkLowerCase } from '@balac/utils'

checkLowerCase('this is lower') // true
checkLowerCase('This is not LOWER') // false
checkLowerCase(123) // false - input is not a string

checkUrlAbsoluteString

检查传入的 URL 字符串参数是否以协议开头来确定它是否是绝对路径，非字符串将返回 false

import { checkUrlAbsolute } from '@balac/utils'

checkUrlAbsolute("http://example.com/page") // true
checkUrlAbsolute("/page") // false
checkUrlAbsolute(123) // false - input is not a string

toCompactCaseString

从传入字符串参数中删除所有空白字符。

import { toCompactCase } from '@balac/utils'

toCompactCase('this is text') // thisistext
toCompactCase('someTEXT') // someTEXT

toLettersString

从传入的字符串参数中删除所有非字母字符

import { toLetters } from '@balac/utils'

toLetters('this is text') // thisistext
toLetters('123-Text!!!') // Text
